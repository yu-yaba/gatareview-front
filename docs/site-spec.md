# ガタレビュ サイト仕様書

対象: 現行実装ベースの簡易仕様  
更新日: 2026-03-20

## 1. サイト概要

ガタレビュは、新潟大学の授業レビューを共有するサイトです。  
シラバスだけでは分かりにくい授業の実態を、学生レビューとして蓄積・閲覧できます。

- フロントエンド: Next.js 13 / TypeScript
- バックエンド: Rails 7 API
- 主な対象ユーザー: 新潟大学の学生
- 利用料金: 無料
- 閲覧は基本的に公開。ログインで一部機能を解放

## 2. 主な画面

- `/`
ホーム。キーワード検索、学部選択、総レビュー数表示、最新レビュー4件、人気授業4件、レビュー未投稿授業4件を表示します。

- `/lectures`
授業一覧。授業名・講師名のキーワード検索、学部絞り込み、並び替え、新しい順・レビュー件数順・評価順に対応します。1ページ20件です。

- `/lectures/[id]`
授業詳細。授業情報、平均評価、レビュー件数、レビュー一覧、ブックマーク、ありがとう、報告、本人レビューの編集を扱います。

- `/lectures/[id]/review`
対象授業へのレビュー投稿フォームです。

- `/reviews/new`
レビュー対象の授業を検索して選ぶための画面です。選択後に対象授業の投稿画面へ遷移します。

- `/mypage`
ログインユーザー向けのマイページです。プロフィール、投稿数、もらったありがとう数、レビュー投稿数ランキング、最近の投稿、最近のブックマークを表示します。

- `/mypage/reviews`
自分の投稿レビュー一覧です。1ページ10件で、編集・削除ができます。

- `/mypage/bookmarks`
自分のブックマーク一覧です。1ページ10件で、授業詳細への遷移とブックマーク解除ができます。

- `/auth/signin`
Googleログイン画面です。ログイン状態を記憶するオプションがあります。

- `/faq`, `/terms`, `/privacy`
FAQ、利用規約、プライバシーポリシーです。

- `/offline`
オフライン時の案内ページです。

## 3. 検索・一覧仕様

- 授業検索は授業名または講師名の部分一致です。
- 学部は別パラメータで絞り込みます。
- 詳細検索は以下に対応します。
授業を受けた年、開講時期、教科書、出席確認、採点方法、単位取得難易度、内容充実度。
- 授業一覧の並び替えは以下です。
最新レビュー順、レビュー件数順、平均評価順。
- ホームの「人気授業」はレビュー件数上位4件です。
- ホームの「レビュー未投稿授業」はレビューがない授業からランダム4件です。
- ホームの「最新レビュー」は新着4件です。

## 4. 認証・権限

- 認証は Google OAuth を使用します。
- フロントは NextAuth、バックエンドAPIは独自JWTで認証します。
- 未ログインでもホーム、授業一覧、授業詳細、FAQ等は利用できます。
- ログイン必須の主な機能は以下です。
マイページ閲覧、ブックマーク、ありがとう、本人レビューの編集・削除。
- バックエンドには管理者専用APIがあります。
授業作成、レビュー期間の作成・更新・有効化・無効化です。

## 5. レビュー仕様

- レビュー投稿は匿名でも可能です。ログイン中に投稿した場合はユーザーに紐づきます。
- 同一ログインユーザーは、同じ授業に複数レビューを投稿できません。
- 必須項目は総合評価とコメントです。
- コメントは30文字以上1000文字以内です。
- 任意項目は以下です。
授業を受けた年、開講、教科書、出席確認、採点方法、単位取得難易度、内容充実度。
- 投稿時は reCAPTCHA による検証を行います。
- 投稿成功後は授業詳細へ戻ります。
- 本人レビューのみ編集・削除できます。

## 6. レビュー閲覧制御

- 授業詳細のレビュー全文閲覧には制限があります。
- 基本条件は「ログイン済み」かつ「レビュー投稿実績1件以上」です。
- さらにバックエンドではレビュー期間機能があり、アクティブ期間が設定されている場合は、その期間内の投稿数で判定します。
- アクティブ期間がない場合は累計投稿数で判定します。
- 権限がない場合でも、授業詳細では先頭レビューのみ全文表示します。
- 2件目以降のコメントは先頭30文字まで表示し、続きはぼかし表示します。
- ホームの最新レビューはログイン状態に関係なく全文表示します。

## 7. リアクション・補助機能

- ブックマーク
ログインユーザーは授業を1件ずつブックマークできます。重複ブックマークは不可です。

- ありがとう
ログインユーザーはレビューに「ありがとう」を送れます。1レビューにつき1回までで、自分のレビューには送れません。

- 報告
レビューの報告は Googleフォームへ遷移します。

- お問い合わせ・削除依頼
フッターから Googleフォームへ遷移します。

## 8. マイページ仕様

- 表示内容は以下です。
ユーザー基本情報、投稿レビュー数、もらったありがとう数、最新レビュー、レビュー投稿数ランキング、最近のレビュー、最近のブックマーク。
- レビュー投稿数ランキングは、レビューを持つユーザーの中での順位を動的に計算します。
- 投稿レビュー一覧では、授業情報付きで自分のレビューを管理できます。
- ブックマーク一覧では、授業ごとの平均評価とレビュー件数も表示します。

## 9. PWA・公開設定

- PWAマニフェストを持ち、ホーム画面追加に対応しています。
- インストールプロンプトを表示します。
- オフラインページを用意しており、キャッシュ済みページの再利用を想定しています。
- SEO向けに `metadata`、OGP、構造化データ、サイトマップを設定しています。

## 10. 主要データモデル

- `Lecture`
授業の基本情報。

- `Review`
評価、コメント、授業関連の詳細項目を持つレビュー。

- `User`
Googleログインユーザー。

- `Bookmark`
ユーザーと授業の保存関係。

- `Thank`
ユーザーからレビューへの感謝リアクション。

- `ReviewPeriod`
レビュー閲覧権限判定に使う期間設定。

- `UserReviewPeriodCount`
各ユーザーの期間別レビュー投稿数。
